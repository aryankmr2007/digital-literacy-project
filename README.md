##### Project Overview

This repository contains my portfolio for the CSE0001 Digital Literacy course. As a Student Digital Ambassador, the goal of this project is to help peers navigate the digital world effectively, from creating professional online profiles to communicating safely and responsibly online. All deliverables across the five course modules are organized within this repository.
